## 🚨 COMMON MISTAKES (AVOID KARO)

❌ **Galti 1:** Name me extra space
```
Wrong: "FTP_HOST " (space hai end me)
Right: "FTP_HOST" (no space)
```

❌ **Galti 2:** Capital letters galat
```
Wrong: "ftp_host" (lowercase)
Right: "FTP_HOST" (UPPERCASE)
```

❌ **Galti 3:** FTP_DIR me ending slash nahi
```
Wrong: "/home/lmssifututorla/learnest-frontend.tutorla.tech"
Right: "/home/lmssifututorla/learnest-frontend.tutorla.tech/"
       (ending me / slash)
```

❌ **Galti 4:** Password me typo
```
Wrong: "Admin@2026+" (ek + kam)
Right: "Admin@2026++" (do ++)
```
